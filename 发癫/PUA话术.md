# PUA话术

```
其实，我对你是有一些失望的。当初进公司，是基于你当时的水平的。
我是希望进来后，你能够拼一把快速变成团队领导者的。
我们这个组，不是把事情做好就可以的。你需要有体系化思考的能力。
你做的事情，他的价值点在哪里？你是否作出了壁垒形成了核心竞争力？
你做的事情，和其他人的差异化在哪里？
你的事情，是否沉淀了一套可复用资料和方法论？
为什么是你来干这个工作，其他人不能干吗？
你需要有自己的判断力， 而不是我说什么你就做什么。
我不需要中规中矩的答卷，它是有一些差异化在里面的。
后续，把你的思考沉淀到组会里，我希望看到你的思考，而不仅仅是进度。
另外，提醒一下，你的产出在我们组是有些单薄的。
如果再这样下去，我不能保证你按时退休。
```

```
先跟大家讲清楚，你们不是在给公司打工，你们是在给自己的未来打工。公司只是一艘船，你们的人生才是目的地。你们现在每一天加班到深夜，每一行代码、每一份报表、每一次熬夜，其实都是在投资你们自己的履历和能力。等到有一天你们站在更高的平台上回头看，你们会感谢今天还愿意拼命的自己。
很多同学总在抱怨，说节奏太快、压力太大、需求变来变去、方案推翻重来。我想说的是：没有被压榨的职场，只有没有成长的打工人。外面有的是朝九晚五、不痛不痒的工作，你们真要的是那种一眼望到头、一辈子拿死工资的人生吗？你们来这儿，是为了突破，是为了对得起自己的天赋和野心。
我知道大家最近很累，连续几个版本迭代、周末也在上线、半夜还在排查 bug。但你们要明白，真正决定你上限的，不是你现在加了多少班，而是你在这个过程中有没有把自己打磨成“不可替代的人”。如果你只是按时完成任务，那你永远只是一个螺丝钉；只有当你开始替别人思考、替团队思考、替公司思考的时候，你才真正开始拥有话语权。
有同学说，“领导，你能不能别总喊我们加班，多招点人不行吗？” —— 我想问你，你是不是只把自己当「资源」而不是「合伙人」？真正的核心骨干，是不会用“多招几个人”来给自己找借口的，而是会想：这件事我能不能先扛下来，把它做成方法论，再去带新人，把我的经验沉淀成流程和标准。
别动不动就拿“工作与生活平衡”来说事。在你没有把自己做成一个有稀缺价值的人之前，谈平衡就是在逃避成长。你羡慕别人自由，是因为别人先用十年的不自由，换来了现在的底气。你现在下班比别人早，未来被市场淘汰也会比别人早。
还有人觉得委屈：为什么努力了这么久，还是看不到立刻的升职加薪？我要很负责任地告诉大家：我们给你的不是立刻兑现的结果，而是一个把你推向极限的环境。你在这儿三年，可能抵得上别人外面干十年。你要的到底是短期的舒服，还是长期的成长？
有人说这叫“压榨”，我不这么看。真正的压榨，是你明明有能力，却被安排做一辈子简单重复的事情；而我们是在逼你，把你潜力里最后一点油都榨出来，帮你看到一个连你自己都不曾想象过的高度。你现在觉得痛苦，是因为你的认知还停留在“打卡领工资”的阶段。
我也知道，这一路上你们会有情绪，会有抱怨，会有想放弃的时刻。但请记住：情绪不值钱，结果才值钱。市场不会因为你累了就对你温柔一点。当你们熬过这段最辛苦的时光，你们简历上的每一行经历、你们做过的每一个项目，都会成为你们未来议价的筹码。
你们现在看的是眼前的一点点加班费、调薪幅度，是这次绩效拿A还是B；但我希望你们看到的是更长远的东西——当你离开这里的时候，你是不是能拍着胸口说一句：“我在这段时间，把自己从一个普通人，升级成了真正能独当一面的职业人。”
所以从今天开始，不要再问“为什么又是我”“为什么要加班”“为什么别人可以准点走”。真正有格局的人，会问的是：“这个事情为什么交给我？我怎么做才能做得比预期更好？我能不能在解决问题的同时顺便升级一下整个团队的能力？”
你们要学会对自己“狠”一点，对自己的时间“贪”一点。趁你们还年轻，趁你们还有被折腾的资本，多扛一点责任，多接一点难事。等哪一天，你发现没有人再愿意为难你、挑战你，才是真正可怕的——说明别人已经不再把你当可以培养的核心了。
最后我想说的是：如果你只是想要一份安稳的工作，那这里可能真的不适合你；但如果你想要一段别人无法复刻的经历、一段可以写进人生履历的高光时刻，那就把这段时间当成你和自己较真儿的修行。
别再问公司能给你什么了，先问问自己，你到底想成为什么样的人，你配不配得上你嘴上说的“梦想”？
```
