# 哥们你记住

```
哥们你记住，你的肾，肺都有俩，但你有一个就能活，运钞车武警第一发都是空包弹，人们都在银行存钱，
屁股不止可以拉屎，你输了只是把钱存进去，但是你放弃了就是真的输了，人家都说十赌九输，
你现在已经九输了。你再博一把就可以收回来，如果你还想东山再起我还看得起你，你现在放弃了我就看不起你，
你记住一颗肾，一个角膜，一个肺都能活偏偏上帝给了你两个，兄弟如果你说你需要200最后一舞，我会毫不犹豫发给你，
但是你跟我说借200坐车要去广东上班，对不起我的钱不会借给懦夫，你太让我失望了。人要相信运气，放平心态，
父母的养老金，家里的房子，老婆的首饰，孩子的学费，朋友的转账，网贷高利贷，都是咱们上岸的资本，
翻身打一场漂亮仗，加油兄弟哪有小孩天天哭，哪有男人天天输，你要不要好好想想人为什么要长两个肾？
父母的养老费和家里的房产证都是翻身的资本，我相信你能做得到
```
