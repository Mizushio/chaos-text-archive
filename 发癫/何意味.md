# 何意味

```
坐拥千亿意味的爷爷去世了，就在家族齐聚准备分遗产时，
律师突然沉声问道：“何意味？”全场霎时死寂。大伯猛地拍桌：“何意味？”
小姑挪瑜道：“何意味？”二伯瞥向他儿子：“何意味？”堂哥轻蔑一笑：“何意味？”
最后，所有目光落在我身上。我后背发凉， 正想隐藏屏保上的何意味，
堂哥突然阴恻恻开口：“何意味？”我慌忙否认：“何意味？！”他不依不饶，
当场找了一张纸写了何意味逼问我：“何意味？”我想都没想就答道：“何意味！”“何意味？！”
整个大厅骤然响起一片倒抽冷气的声音。我瞬间僵住—一何意味！亲戚们脸上顿时浮出毫不掩饰的讥嘲，
似乎在说“何意味…”谁知，律师却大步走向我，“啪”地朝我鞠躬道：“何意 味。”
全场炸了。大伯疯了一样揪住律师嘶吼：“何意味?!!”小姑尖叫：“何意味?!?”
律师掏出一份金光闪闪的遗嘱补充协议，朗声宣读：“何意味。何意味?何意味，何意味!何意味。”
我原本弯着的腰，一寸一寸地挺直。苍白的脸浮起血色，眼中何出前所未有的意味。
接着，我一把扯开外套，露出印着“何意味”的T 恤，声如洪钟：“何一一意味！？”
全场亲戚瞬间腿软瘫倒。下一秒，所有人连滚带爬掏出纸笔，颤抖着写下何意味，
谄笑道：“何意味！”“何意味？！”“何意味？”
```
