# 居然给三角洲充了6块

```
父母居然给三角洲充了6块😭我知道的时候气的都手抖，当时感觉天都塌下来了，
辛辛苦苦养他们这么多年感觉努力都白费了， 看到手机里弹出的6元扣费通知时，
我的手指突然不听使唤地颤抖起来。那串冰冷的数字在屏幕上闪烁，像一把锋利的刀片，
生生划开了我这些年来筑起的全部期待。 可我的眼前却一阵阵发黑。🤧6块钱——这个微不足道的数字此刻重若千钧，
压得我几乎喘不过气。 这些年凌晨五点起床准备的早餐，深夜台灯下写作业的疲倦，省吃俭用给他们报的各类保险，
所有付出突然都变成了可笑的独角戏。我甚至能听见自己心脏碎裂的声响，就像阳台上那盆被他们失手打碎的多肉，
精心培育的成果转眼就成了一地狼藉。我突然想起上周他们信誓旦旦保证"绝对不冲游戏"时亮晶晶的眼睛。
🥺灶台上炖着的排骨汤咕嘟咕嘟冒着泡，香气弥漫了整个厨房，可我只闻到一阵阵令人作呕的铜锈味。 
储蓄罐里攒了半年的硬币在哗啦作响，那是准备给他们买生日礼物的。现在它们都在嘲笑我，笑我这个孩子当得多失败。
墙上的全家福里，他们搂着我脖子笑得多甜啊，怎么转眼就能面不改色地按下支付密码

```
