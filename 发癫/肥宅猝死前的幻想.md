# 肥宅猝死前的幻想

```
有一次我去面试，面试官问了我好几个问题，我太紧张了回答的不好，正当我以为面试失败的时候，突然电话响了，
我拿起了手机，面试官一眼就看到我的手机壳是二次元角色，突然两眼放光，问我看了多少部番，不怎么看也就56百部吧，然后他着急忙慌的把他们老板叫来了，
原来老板是个小二次元，求我带带他，我就勉为其难的带带了，谁能想到身价几干万的老板居然连咒术回战都没有看过，然后我带老板看了鬼灭之刃，
看了间谍过家家 看了蓝锁 老板开心坏了，直接给我分了一半的公司股份，一时间整个公司都在讨论我，说我是天才少女啥的，都让我每天给他推荐动画，
说我是二次元高手，我都不知道怎么办才好了
```
